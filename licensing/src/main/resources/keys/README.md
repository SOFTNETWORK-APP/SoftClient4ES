# License Public Keys

Ed25519 public keys in JWK JSON format (`.jwk` files).

## Format

```json
{
  "kty": "OKP",
  "crv": "Ed25519",
  "x": "<base64url-encoded public key>",
  "kid": "<key-id>"
}
```

## Key Naming

Files are named `{kid}.jwk` where `kid` matches the JWT `kid` header.
Example: `softclient4es-2026-03.jwk`

## Adding Keys

Production keys are generated by the license server and added here during release.
